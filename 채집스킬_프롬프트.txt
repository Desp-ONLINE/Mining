구현할 내용 (채광과 동일):

1. 레벨/경험치: 채집 1회마다 경험치 지급, 레벨업 시 스킬 포인트 지급.
   최대 레벨/레벨당 필요 경험치/포인트 지급량은 MongoDB 컬렉션에서 관리
   (예: GatheringLevelConfig / GatheringSkillConfig), 문서 없으면 기본값 자동 시드.

2. 스킬 3종 (레벨별 수치/포인트 비용은 DB에서 관리):
   - 피로도 무효: 일정 확률로 채집 시 피로도 증가 안 함
   - 더블 드랍: 일정 확률로 드랍 아이템 2배
   - 희귀 채집물: 지정된 아이템 목록의 드랍 가중치를 n% 올리고 나머지를 n% 내려서
     전체 확률 합 100%를 유지하는 방식 (채광의 getRandomDropItem 참고)

3. 플레이어 데이터: 기존 DTO에 level, exp, skillPoints, skills(Map) 추가하고
   DB 로드/저장에 반영. 주의: 피로도 증감 시 새 DTO를 만들면 스킬 데이터가
   날아가니까 기존 DTO를 setter로 수정하는 방식으로 (채광에서 고친 버그임).

4. 명령어 /채집스킬: 인자 없으면 스킬 GUI 열기, "투자 [스킬] (레벨수)"로 포인트 투자,
   "리로드"는 OP 전용으로 DB 설정 다시 로드. plugin.yml에 한/영 오타 별칭도 등록.

5. 연출: 액션바에 드랍 아이템 + 레벨/경험치 표시, 스킬 발동 시 타이틀 메시지,
   레벨업 시 채팅 메시지 + 레벨업 사운드. 색상은 채광이랑 같은 파스텔톤.

6. PlaceholderAPI: level, exp, required_exp, skill_points, skill_<스킬명> 추가.
